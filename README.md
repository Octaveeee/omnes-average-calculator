# Omnes Average Calculator

A userscript for Tampermonkey that enhances the Omnes Education campus platform by calculating student grade averages.

## What it does

This userscript automatically calculates and displays the average of your grades on the Omnes Education campus platform. It provides a convenient way to track your academic performance directly on the platform.


## Installation

### Prerequisites

You need to have **Tampermonkey** installed in your browser:

- **Chrome/Edge**: Install from [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- **Firefox**: Install from [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/tampermonkey/)
- **Safari**: Install from [Safari Extensions](https://apps.apple.com/app/tampermonkey/id1482490089)

### Step-by-step installation

1. **Open Tampermonkey Dashboard**
   - Click on the Tampermonkey icon in your browser toolbar
   - Click on "Dashboard"

2. **Create a new script**
   - Click on "Create a new script..."
   - Delete all the default code

3. **Copy the script**
   - Open the `average-calculator.js` file from this repository
   - Copy all its contents
   - Paste it into the Tampermonkey editor

4. **Save the script**
   - Press Ctrl+S
   - The script should now appear in your Tampermonkey dashboard

5. **Verify the script is enabled**
   - In the Tampermonkey dashboard, make sure the toggle next to "Omnes Average Calculator" is green (enabled)

6. **Test the installation**
   - Visit your grades page
